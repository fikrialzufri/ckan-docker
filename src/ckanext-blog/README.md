# ckanext-blog

Extension CKAN untuk menambahkan fitur blog/post.

## Fitur

- Membuat, membaca, mengupdate, dan menghapus post blog
- Daftar semua post blog
- Template yang responsif

## Instalasi

Extension ini sudah terintegrasi dengan setup Docker. Pastikan plugin `blog` ditambahkan ke `CKAN__PLUGINS` di file `.env`.
